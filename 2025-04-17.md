## 🌱 Day 11 - `git pull`

### 🔍 学んだこと
- `git pull` = `git fetch` + `git merge`
- トラッキングブランチを元にpull先が決まる
- `git pull origin main` のように明示的にも書ける
- `main` じゃないときはブランチ確認＆切り替えを行う

### ❓ 今日の質問と学び
- `git pull origin main` の意味と、トラッキングブランチについて
- `main` じゃないときの対応方法

### 🛠 実行コマンド
```bash
git branch
git checkout main
git pull
